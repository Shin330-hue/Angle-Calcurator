#### ストア掲載情報

**簡単な説明（80文字以内）**
```
金属加工・建築現場で使える直角三角形計算機。辺と角度を入力するだけで即座に計算。
```

**詳しい説明**
```
【アプリ概要】
三角形計算機は、金属加工や建築現場で必要な直角三角形の計算を素早く行えるアプリです。辺の長さや角度を入力するだけで、三角関数を使って未知の値を自動計算します。

【こんな方におすすめ】
・金属加工の現場で角度計算が必要な方
・建築・土木作業で斜面や勾配を計算する方
・DIYで正確な角度を出したい方
・三角関数の学習に活用したい学生

【主な機能】
■ 直角三角形の自動計算
- 3つ以上の値（辺1つ以上を含む）を入力すると、残りを自動計算
- 三平方の定理、正弦・余弦・正接を活用

■ わかりやすい図形表示
- 入力値がリアルタイムで図形に反映
- 各辺・各角度の位置が一目でわかる

■ 表示精度のカスタマイズ
- 角度・辺の長さの小数点以下桁数を0〜5桁で設定可能
- 現場の要求精度に合わせて調整できます

■ 計算結果の保存機能
- よく使う三角形を名前をつけて保存
- 保存した値はいつでも呼び出し可能

【使い方】
1. 計算機画面で辺の長さや角度を入力
2. 「計算」ボタンをタップ
3. 未知の値が自動で計算されます

【対応端末】
Android 7.0 以上

【プライバシー】
本アプリはインターネット接続不要で動作し、入力データは端末内にのみ保存されます。
```
